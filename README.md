# Hermes

Hermes, Home Assistant üzerinde çalışan çok siteli bir fiyat takip add-on'udur.

Takip edilen alanlar:
- `products`: Amazon, Hepsiburada, Trendyol, Network ürün linkleri
- `amazon_search_pages` + `amazon_search_targets`: Amazon arama tabanlı takip

Bildirimler Pushover üzerinden gönderilir.  
Ingress paneli üzerinden durum, özet tablo ve test bildirimi yönetilebilir.

## Home Assistant Repository

Home Assistant > Add-on Store > Repositories alanına:

`https://github.com/nriacr/hermes`

ekleyerek kurulabilir.
